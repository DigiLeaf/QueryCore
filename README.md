# QueryCore - An Offline Search Engine

A C++ project that indexes local text files and supports offline searching using
an inverted index, boolean queries, and TF-IDF ranking.

## Features (Planned)
- File crawling using std::filesystem
- Tokenization and text normalization
- Inverted index construction
- Boolean AND/OR queries
- TF-IDF ranking
- Command-line interface

## Project Structure
- src/: source files
- include/: header files
- data/: test documents
